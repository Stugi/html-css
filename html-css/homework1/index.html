Часто задаваемые вопросы

Ошибочно отправили документ(извещение)?

Ваш начальник сектора(отдела) может разблокировать документ (отменить отправку)
через программу "Администрирование программы "Журналы"

Инструкция к программе "Администрирование программы "Журналы"

Как заблокировать документ?

Доступны два способа:

  1. Начальник сектора(отдела) может заблокировать документ
    через программу "Администрирование программы "Журналы"
  2. Исполнитель КД может заблокировать свой документ через прграмму "Журналы"
